# 相关技术
| 技术/框架         | 核心说明                                                                 |
|-------------------|--------------------------------------------------------------------------|
| Spring Boot       | 快速开发Spring应用的脚手架，简化配置、依赖管理与项目部署                 |
| Redis             | 高性能键值对数据库，此处用于**主键回显**场景（如分布式ID生成后返回）|
| MyBatis           | 半自动ORM框架，支持自定义SQL，灵活适配复杂数据库操作                     |
| MySQL             | 关系型数据库，作为业务核心数据存储载体                                   |
| Nginx             | 高性能反向代理服务器，支持负载均衡、静态资源托管、接口转发等             |
| Spring MVC        | 基于MVC模式的Web框架，处理HTTP请求、参数绑定、视图渲染等                 |
| AliOss            | 阿里云对象存储服务，用于存储图片、视频等非结构化文件                     |
| AOP（面向切面编程） | 面向切面编程思想，统一处理日志、权限、事务、异常等横切关注点             |
| Lombok            | 代码简化工具，通过注解自动生成getter/setter、构造方法等，减少冗余代码     |
| 内网穿透          | 突破内网限制，将本地服务暴露至公网，便于外网访问场景调试（如微信回调）|
| Spring Cache      | 基于注解的缓存抽象框架，核心特性：<br/>1. 注解化使用：仅需添加注解即可实现缓存能力<br/>2. 多缓存适配：底层可切换EHCache、Caffeine、Redis等实现<br/>3. 前置条件：需导入对应Maven坐标<br/>4. 核心注解：<br/> - @EnableCaching：开启缓存注解功能（通常加在启动类）<br/> - @Cacheable：方法执行前查缓存，有则返回，无则执行方法并缓存结果<br/> - @CachePut：将方法返回值存入缓存（不拦截方法执行）<br/> - @CacheEvict：从缓存中删除指定数据 |
| Spring Task       | Spring内置定时任务工具，无需依赖第三方组件，通过注解快速实现定时/周期性任务 <br/> cron表达式|
| WebSocket         | 基于**TCP**的一种新的网络协议，实现浏览器和服务器全双工，一次握手就创建持久连接，并进行双向数据传输，建立的是长连接 <br> HTTP是基于TCP的短连接，需要三次握手建立连接，是“请求—响应”式的，不是全双工 <br> 一般实时更新的，用的就是WebSocket |
| Apache ECharts     |**前端技术** <br> 基于JS的数据可视化图标库，提供个性化定制的数据可视化图表|
| Apache POI | 一个处理Microsoft Office各种文件格式的开源项目，利用POI可以在Java程序中对M.O.各种文件进行读写操作（一般都是操作Excel文件）|
